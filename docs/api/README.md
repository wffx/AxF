# API 文档

本目录用于存放主工程各组件的 API 参考文档，和普通设计说明、组件说明文档区分开。

约定：

- `docs/api/<component>/` 存放对应组件的命令行接口、Python API、服务接口或数据接口说明。
- `docs/<component>/` 存放组件定位、使用背景、集成说明和非 API 类文档。
